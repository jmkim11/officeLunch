# Recommendation MVP 리뷰 체크리스트

## 도메인
- ACTIVE 상태에서만 재추천과 선택이 가능한가?
- 선택 완료 후 상태 변경이 차단되는가?
- 추천 이력이 중복 저장되지 않는가?

## 트랜잭션
- 세션 생성과 첫 추천 이력 저장이 하나의 트랜잭션이어야 하는가?
- 외부 장소 API가 연결되면 호출을 트랜잭션 밖으로 분리해야 하는가?

## 동시성
- 동일 세션에 재추천 요청이 동시에 들어오면 같은 식당이 선택될 수 있는가?
- 유니크 제약조건 충돌을 사용자 응답으로 어떻게 변환할 것인가?

## 면접 설명
- 추천 세션과 이력을 분리한 이유를 설명할 수 있는가?
- Redis를 넣지 않은 이유를 설명할 수 있는가?
- AI가 작성한 코드 중 직접 수정 가능한 범위는 어디까지인가?
